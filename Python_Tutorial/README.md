# Learn Python - Scratch to Advanced
## Your Structured Learning Path

### How to Use This
Go folder by folder in order. Each folder has theory + practice files.

### Learning Order:
1. 01_BasICS - Start here
2. 02_Control_Flow - Making decisions in code
3. 03_Functions - Reusable code blocks
4. 04_Data_Structures - Organizing data
5. 05_OOP - Object-Oriented Programming
6. 06_Projects - Build real things

### For Each Topic:
1. Read the theory file
2. Run the example file
3. Do the exercise file
4. Check the solution only after trying

### Tips:
- Practice daily (30 mins is enough)
- Type code yourself, don't just read
- Make mistakes and fix them
- Build small projects as you learn
