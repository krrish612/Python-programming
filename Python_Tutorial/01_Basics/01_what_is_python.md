# What is Python?

## In Simple Words
Python is a language computers understand. Like English or Hindi, Python has rules (grammar) but it's for talking to computers.

## Why Learn Python?
- Easy to read and write
- Used in AI, Web, Data Science, Automation
- High demand = Good salary

## Your First Python Code
```python
print("Hello, World!")
```
`print` = tell the computer to show text on screen

## Variables - Named Boxes
Think of variables as labeled boxes where you store things:

```python
name = "Krrish"      # Box labeled 'name' has text "Krrish"
age = 15             # Box labeled 'age' has number 15
is_student = True    # Box labeled 'is_student' has True/False

print(name)          # Shows: Krrish
print(age)           # Shows: 15
```

## Data Types (What Can Go In Boxes)
| Type | Example | Meaning |
|------|---------|---------|
| str (string) | "Hello" | Text |
| int | 15 | Whole number |
| float | 15.5 | Decimal number |
| bool | True/False | Yes or No |
