# Python Quick Cheat Sheet

## Variables
```python
name = "Krrish"      # String
age = 15             # Integer
price = 9.99         # Float
is_active = True      # Boolean
```

## Math Operators
```python
+ - * /            # Basic
//                  # Integer division
%                   # Remainder
**                  # Power
```

## Data Types
```python
str       # Text
int       # Whole number
float     # Decimal
bool      # True/False
list      # [1, 2, 3]
tuple     # (1, 2, 3)
dict      # {"key": "value"}
```

## Comparisons
```python
==  # Equal
!=  # Not equal
>   # Greater
<   # Less
>=  # Greater or equal
<=  # Less or equal
```

## Logical
```python
and  # Both true
or   # Either true
not  # Flip result
```

## Loops
```python
for i in range(10):
    print(i)

while True:
    if condition:
        break
```

## Functions
```python
def func_name(param):
    return value
```

## Lists
```python
list.append(item)
list.remove(item)
list.sort()
list.reverse()
```

## Strings
```python
string.upper()
string.lower()
string.replace("old", "new")
string.split(",")
string.join(list)
```

## Print
```python
print("Hello")
print(f"Value: {var}")
print("Formatted: {:.2f}".format(num))
```

## Input
```python
name = input("Enter name: ")
age = int(input("Enter age: "))
```
