# Math Operations in Python

## Simple Math Like a Calculator

Python can do math. Think of it as a smart calculator.

## Basic Operations

| Symbol | Meaning | Example | Result |
|--------|---------|---------|--------|
| + | Add | 5 + 3 | 8 |
| - | Subtract | 10 - 4 | 6 |
| * | Multiply | 4 * 3 | 12 |
| / | Divide | 15 / 3 | 5.0 |
| // | Whole Division | 17 // 5 | 3 |
| % | Remainder | 17 % 5 | 2 |
| ** | Power | 2 ** 3 | 8 |

## Examples in Code

```python
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333...
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000
```

## Shorthand (Same Thing)

```python
x = 5
x = x + 3    # Same as: x += 3
x += 3       # Now x = 8
x -= 2       # Now x = 6
x *= 2       # Now x = 12
```
